# 误差分析

## 有效结论

- 领域 LoRA 相比同一 Qwen2.5-1.5B 底座，提高了根因提及率、证据覆盖率、格式合规率和 ROUGE-L。
- LoRA 输出中的传感器 ID 均来自输入证据，自动检测到的传感器幻觉率为 0%。
- Qwen2.5-0.5B 能模仿诊断报告结构，但没有在测试事件中正确复述参考根因或关键证据。

## 主要失败案例

- LoRA 在测试样本 1 中选择了候选 `1_MV_001_STATUS`，未覆盖参考答案的目标根因，说明它仍可能被较高的节点分或关系退化分带偏。
- 未微调 1.5B 在两个事件中没有覆盖参考根因，并在部分回答中生成了输入中不存在的传感器 ID。
- 0.5B 弱基线倾向于解释“联合分、节点分、边退化”等概念，而不是结合具体传感器完成诊断。

## 结果限制

- 测试集只有 4 个事件；每个事件会造成 25 个百分点的波动。
- 参考答案由项目现有固定模板生成，因此本实验主要验证结构化证据归纳和报告格式学习，不等同于专家标注的独立根因诊断评测。
- 本轮受时间限制没有下载和运行跨家族模型，外部对比仅包含同系列 0.5B 弱基线。
- 自动幻觉率只检查传感器 ID，没有完整判断自然语言工艺描述或数值是否真实。
